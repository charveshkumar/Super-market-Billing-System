Title : Super market Billing System
Project Description : The Python Supermarket Billing System is a simple command-line application developed using Python. It allows users to view available grocery items, select products, enter quantities, calculate the total bill, apply taxes, and generate a detailed invoice. This project is designed for beginners to practice Python fundamentals such as loops, conditional statements, dictionaries, lists, user input handling, and basic billing logic.
Key Features
•	Display available supermarket products and prices.
•	Purchase multiple items in a single session.
•	Quantity validation to prevent invalid inputs.
•	Automatic price calculation based on quantity.
•	Tax calculation (12% GST).
•	Generates a detailed shopping bill.
•	Stores purchased item details in lists.
•	User-friendly command-line interface.
•	Handles unavailable products gracefully.

Technologies Used
o	Python 3
o	Python Built-in Data Structures:
o	Dictionary
o	List
o	Tuple
o	Control Flow:
o	Loops (while)
o	Conditional Statements (if-elif-else)
o	Input/Output Operations
